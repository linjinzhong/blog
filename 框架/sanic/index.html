简介

  1. 是什么
    Sanic 是 Python Web 服务器和 Web 框架，旨在提高性能。
    Sanic 不仅仅是一个 框架，它还是一个 Web 服务器，具备开箱即用的功能。
    它可以用于编写，部署和扩展生产级 Web 应用程序。 🚀
    Build fast. Run fast.
  2. 目标
    提供一种简单且快速，集创建和启动于一体的方法，来实现一个易于修改和拓展的 HTTP 服务
